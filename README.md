# Spring-Utils

[![Build](https://github.com/tommasop1804/spring-utils/actions/workflows/build.yml/badge.svg)](https://github.com/tommasop1804/spring-utils/actions/workflows/build.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Kotlin](https://img.shields.io/badge/Kotlin-2.3.0-blue.svg)](https://kotlinlang.org)
[![JDK](https://img.shields.io/badge/JDK-21-orange.svg)](https://openjdk.org)

A comprehensive Kotlin/JVM utility library providing extensions, helpers, and ready-to-use classes for common development tasks.

## Installation

### Gradle (Kotlin DSL)

```kotlin
repositories {
    mavenCentral()
}

dependencies {
    implementation("dev.tommasop1804:spring-utils:1.3.0")
}
```

### Gradle (Groovy)

```groovy
repositories {
    mavenCentral()
}

dependencies {
    implementation 'dev.tommasop1804:spring-utils:1.3.0'
}
```
